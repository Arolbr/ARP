## 项目名称

Rust ARP Spoofing

## 项目简介
这是一个基于 Rust + pnet 实现的 ARP 协议研究演示工具。
项目完整实现了以下能力：
1. 基于网卡直连的 ARP Request / Reply 构造与发送
2. 局域网内 ARP 扫描，发现在线主机及其 MAC
3. 对目标主机与网关实施 双向 ARP 欺骗
4. 捕获并解析以太网帧与 IPv4 数据包
5. 在攻击结束后 主动恢复 ARP 表，尽量减少网络污染

**该项目并非“玩具级 Demo”，而是一个完整的 ARP 层攻击与转发流程实现，涵盖从链路层到网络层的真实处理逻辑，适用于对以下主题的深入理解：
ARP 协议的工作机制与缺陷
局域网信任模型的脆弱性
Rust 在底层网络编程中的可行性与边界**
### 使用时应该注意, 因为该工具基于Windows编写, 所以使用时需要依赖Npcap, 所以需要单独安装Npcap, 安装包我已经与ARP.exe一起提供

## 功能概览（非操作指引）
1. 网络接口枚举与选择
2. IPv4 网段解析与在线主机探测
3. ARP 欺骗循环发送机制
4. 数据包捕获、判别、重写与转发
5. 原子状态控制与多线程协作
6. Ctrl+C 信号处理与攻击收尾恢复
全程用户态完成。
